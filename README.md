# AI協調開発フレームワーク (AI Collaborative Development Framework)

本プロジェクトは、人間とAIが「設計意図（Intent）」を核に協力し、高品質で持続可能なソフトウェアを開発するためのフレームワークです。

## コンセプト
単なるコード生成ではなく、AIを「熟練のアーキテクト」として扱い、開発の憲法（`ARCHITECTURE_MANIFEST.md`）を共同で設計・運用します。

## 主要ドキュメント
- [DESIGN_PHILOSOPHY.md](./DESIGN_PHILOSOPHY.md): 本フレームワークの根本思想とAIへの基本指示。
